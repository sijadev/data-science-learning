# 📝 Grundlegende Syntax

## Python-Philosophie

Python folgt dem Prinzip "Readability counts" - Code soll lesbar und verständlich sein.

## Einrückungen

Python nutzt Einrückungen statt geschweifte Klammern für Code-Blöcke.

## Kommentare

- `# Einzeiliger Kommentar`
- `"""Mehrzeiliger Kommentar"""`
